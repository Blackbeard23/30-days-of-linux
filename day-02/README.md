## What I Learned

- `man` commands which is short for *manual* helps you to understand other commands
```sh
man <command>
```
- `clear` - clear all previous commands that were ran in the current terminal. shortcut is `ctrl + l`
- `pwd` - whenever you feel lost in the file sytem, call the command to know where you are. It will print the current folder path.
- `ls` - short for *list*, which is used to list all the files that the folder contains. By adding a folder name or path, it will print that folder contents.
```sh
ls /bin
```
